# Aplicacao DevOps

Aplicacao web desenvolvida com Node.js e Express.js para a avaliacao N1 da disciplina de Ferramentas de Implantacao Continua - DevOps.

## Integrantes

- Pedro Alves da Rosa Cardoso
- Gustavo Correia

## Pre-requisitos

- Node.js instalado
- Git instalado

## Como instalar e executar

1. Clone o repositorio: git clone https://github.com/Calangogu1/aplicacao-devops
2. Instale as dependencias: npm install
3. Inicie o servidor: node src/index.js
4. Acesse no navegador: http://localhost:3000

## Endpoints

- GET / : Retorna a pagina de boas-vindas
- GET /integrantes : Retorna a lista de integrantes do grupo

## Tecnologias utilizadas

- Node.js
- Express.js
