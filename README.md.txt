# Aplicação DevOps

Aplicação web desenvolvida com **Node.js** e **Express.js** para a avaliação N1 da disciplina de Ferramentas de Implantação Contínua - DevOps.

## 📋 Pré-requisitos

- [Node.js](https://nodejs.org) instalado
- [Git](https://git-scm.com) instalado

## 🚀 Como instalar e executar

1. Clone o repositório:
```bash
   git clone https://github.com/Calangogu1/aplicacao-devops
   cd aplicacao-devops
```

2. Instale as dependências:
```bash
   npm install
```

3. Inicie o servidor:
```bash
   node src/index.js
```

4. Acesse no navegador: [http://localhost:3000](http://localhost:3000)

## 🔗 Endpoints

| Endpoint | Método | Descrição |
|---|---|---|
| `/` | GET | Retorna a página de boas-vindas |
| `/integrantes` | GET | Retorna a lista de integrantes do grupo |

## 🛠️ Tecnologias utilizadas
